批量FD处理

API 说明

此API用于从一个标准的环境导出FD，然后导入到其余单位

请求参数

source_project_code String 源项目Code
target_project_codes List 目标单位
fd_codes List 需要操作的FDCodes
fd_target_database String fd所属目标数据库

请求示例

{
   "source_project_code":"deyang",
   "target_project_codes":["deyang","yizhou"],
   "fd_codes":["test_code1","test_code2"],
   "fd_target_database":"criminal_business"
}
